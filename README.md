# Food Delivery App

Food delivery mobile application with restaurant browsing, ordering, and real-time delivery tracking.

## Tech Stack

- **Framework**: React Native
- **Language**: TypeScript
- **Navigation**: React Navigation
- **Package Manager**: npm / yarn

## Project Structure

```
food-delivery-app/
├── src/
│   ├── components/
│   ├── screens/
│   ├── navigation/
│   ├── hooks/
│   └── utils/
├── android/
├── ios/
├── .env.example
├── package.json
├── tsconfig.json
└── README.md
```

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. For iOS:
   ```bash
   cd ios && pod install && cd ..
   ```

3. Start the development server:
   ```bash
   npm start
   ```
